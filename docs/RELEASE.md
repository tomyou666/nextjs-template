## バージョン 1.0.0 (2025-03-24)

### 初期リリース

Next.JSテンプレートの初期バージョンをリリースしました。このテンプレートは、モダンなNext.jsアプリケーション開発のための基盤を提供します。

### 主な機能

- **Next.js 15**: App Routerを使用した最新のNext.jsフレームワーク
- **TypeScript**: 型安全なコード開発環境
- **Auth**: 認証機能
- **UI/UXコンポーネント**:
  - shadcn/uiを使用したモダンなUIコンポーネント
  - ダークモード/ライトモードのサポート

- **データ管理**:
  - TanStack Query (React Query): サーバーデータの効率的な管理
  - TanStack Table: 高度なテーブル機能
  - Zustand: シンプルな状態管理

- **フォーム処理**:
  - Conform: 型安全なフォーム管理
  - Zod: スキーマバリデーション

- **レンダリングデモ**:
  - クライアントサイドレンダリング (CSR)
  - サーバーサイドレンダリング (SSR)
  - 静的サイト生成 (SSG)
  - インクリメンタル静的再生成 (ISR)

- **通知システム**:
  - トースト通知
  - アラート通知
  - エラーハンドリング

- **ドキュメント**:
  - 各機能のヘルプダイアログ
  - マークダウン形式の詳細な説明

### 技術スタック

- Next.js 15
- TypeScript
- NextAuth
- TanStack Query & Table
- Zustand
- Conform & Zod
- shadcn/ui
- Tailwind CSS
- Lucide Icons
